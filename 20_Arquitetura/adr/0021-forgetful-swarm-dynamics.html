ADR: Forgetful Swarm Dynamics (The "Bubble" Hypothesis) ​

Context ​

Entities (agents/blobs) that cluster but constantly generate 'conflict signals' (high drift, low stability) effectively create entropic bubbles that drain the pipeline. We need a formal mechanism for 'forgetting' these associations to allow the swarm to stabilize into islands of coherent activity.

The Forgetfulness Protocol ​

  1. Isolation Islands: When a triad of nodes (Agent A, Agent B, Context C) exhibits persistent entropic drift > 0.6 over 3 consecutive enrichment cycles, the association is marked as a 'Conflict Island'.
  2. Forgetting Mechanism: Instead of deleting, we apply a 'Stigmergic Decay' to the relationship links. The weight of the association decays linearly until it falls below the enrichment threshold (causing it to disappear from the L2/L3 synthesis view).
  3. Emergence: This allows for 'vices' (stable conflict cycles) to be visualized as distinct 'islands' in the graph, which can then be synthesized by higher-order L3 agents as distinct behavioral archetypes.

Implementation Path ​

  • Schema Update: Add decay_score to the links table in the database.
  • Agent Policy: Implement a worker agent responsible for the DecayRoutine (running during low-resource periods).
