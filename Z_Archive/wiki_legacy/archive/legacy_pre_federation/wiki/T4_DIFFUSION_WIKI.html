Introdução ​

O NEXUS implementa difusão semântica sobre o grafo T4 como um Personalized PageRank (PPR) com horizonte fixo de 3 hops. A parada antes da convergência evita que a distribuição estacionária π_i ∝ degree(i) domîne o ranking por topologia. O sistema amostra o cone de luz semântico em t=3, não o equilíbrio global.

Arquitetura da Difusão ​

Fluxo: Query → T3 (sementes) → T4 (PPR walk) → T2 (hidratação) → Context

A difusão opera sobre escalares (scores nodais), não vetores. Dimensionalidade 768D intervém apenas no cálculo offline do stress (mesh.rs:38-56), nunca em query time.

Formulação Matemática ​

Condutância da aresta w_q(e) — main.rs:825-827 ​

sql
w = COALESCE(l.stress, 1.0) 
  * EXP(-0.05 * EXTRACT(EPOCH FROM (now() - COALESCE(b.content_date, b.last_ingested_at))) / 86400.0)

Derivação:

w = stress × exp(-0.05 · Δt / 86400)
  • stress: energia Hamiltoniana da aresta (offline, mesh.rs:38)
  • Δt: idade da aresta em segundos
  • exp(-0.05·Δt/86400): decaimento temporal (≈ 0.95 para 30 dias)

w é condutância (admitância) — capacidade de transferência de relevância, não trabalho acumulado.

Fluxo por aresta — main.rs:855-860 ​

rust
flow = (s * PPR_ALPHA * w) / s_deg
*next.entry(tgt).or_insert(0.0) += flow;

Interpretação física: Lei de Ohm discretizada. Score = potencial. Condutância = w. Resistência = deg. α = 0.85: 85% da energia redistribuída por hop; 15% reinjetado nas sementes.

Hamiltonian de stress — mesh.rs:38-56 ​

rust
H = max(0, ||V_t - T⊙V_s||² / N - δ) × √(M_s × M_t) × w
// M = 10^level  (L0=1, L4=10000)
// N = 768  (nomic-embed-text)
// δ = damping_factor

Stress alto indica contradição semântica. Stress baixo indica coerência. A difusão flui por arestas de baixo stress.

Análise de sensibilidade de δ (fator de amortecimento) ​

  • Faixa válida: δ ∈ [0, 4/N] = [0, 0.0052] para N = 768 vetores unitários (FORMAL_ANALYSIS_HAMILTONIAN.md §6.3)
  • Se δ ≥ δ* para uma aresta: H = 0 (aresta "silencia")
  • Se δ é uniforme e alto demais: energia global do grafo colapsa para zero
  • δ deve ser calibrado empiricamente após população do T3 (id., §8.3)

Três Mecanismos de Limitação ​

Mecanismo 1: Horizonte fixo de hops ​

rust
const PPR_MAX_HOPS: usize = 3;   // main.rs:24
for _hop in 0..PPR_MAX_HOPS {   // main.rs:812

s_k+1(t) = Σ_{s→t} s_k(s) · α · w(s,t) / deg(s)

Após 3 hops: α³ ≈ 0.614. Contribuição marginal de hop 4 cai abaixo do ruído. Custo: O(n · d²) — polinomial, não exponencial.

Mecanismo 2: Atenuação temporal exponencial ​

sql
EXP(-0.05 * EXTRACT(EPOCH FROM (now() - COALESCE(b.content_date, b.last_ingested_at))) / 86400.0)

Decaimento lento no curto prazo; cumulativo por hop — arestas >20 dias perdem ~5% de condutância por salto.

Mecanismo 3: Poda por limiar ​

rust
next.retain(|_, v| *v &gt;= PPR_MIN_SCORE);  // main.rs:866
const PPR_MIN_SCORE: f64 = 0.01;           // main.rs:25

Nós com score acumulado abaixo de 0.01 são eliminados do subgrafo ativo. O grafo efetivo de difusão é menor que o grafo completo.

A Escolha de 3 Hops: Geodésica vs. Equilíbrio ​

Contraste formal ​

FormulaçãoExpressãoDominaResultado
Horizonte limitado (NEXUS)s_3 = P³·s_0Query (sementes)Vizinhança semântica
Equilíbrio (PPR padrão)s* = lim_{k→∞} P^k·s_0 = πTopologia (grau)π_i ∝ degree_i

Prova de dominação por grau no equilíbrio ​

A distribuição estacionária π satisfaz π = πP. Para grafo não-dirigido, a transição estocástica é P(i,j) = A(i,j) / deg(i). O balanceamento detalhado (condição necessária e suficiente para reversibilidade) exige:

π_i · P(i,j) = π_j · P(j,i)

Substituindo a definição de P:

π_i · A(i,j) / deg(i) = π_j · A(j,i) / deg(j)

Como A(i,j) = A(j,i) para arestas não-dirigidas:

π_i / deg(i) = π_j / deg(j) = C   (constante para todo par conectado)

Portanto:

π_i = C · deg(i)    onde C = 1 / Σ_k deg(k)

Implicação: qualquer PPR que itere até convergência retorna π_i ∝ degree(i) — a query é diluída na topologia global. Os nós mais conectados dominam, independentemenete da semente. Para um grafo scale-free, o hub recebe massa proporcional a k onde k é seu grau, swamping qualquer sinal de seed.

Por que 3 hops preserva localidade ​

A partir de 3 hops, α³ ≈ 0.614. O score acumulado de qualquer caminho adicional é menor que o limiar de ruído (0.01). O sistema para antes da mistura global — o contexto ainda carrega direção informacional da seed.

Interpretação como cone de luz: 3 hops define o cone de luz semântico das sementes. Nós fora do cone são inacessíveis em tempo finito. Dentro do cone, o caminho mais relevante é a geodésica no espaço de similaridade, não o random walk.

Dimensão de Hilbert: Por Que N=768 Não É Computado em Query ​

O espaço de Hilbert intervém em duas camadas distintas:

CamadaDimensionalidadeQuando intervém
T3 (Qdrant)768Donly no cálculo de stress offline
T4 (PPR)escalarquery time — escalares, não vetores

Nunca em query time: difusão PPR opera sobre scores nodais (escalares). O espaço de Hilbert 768D nunca é iterado explicitamente durante a difusão. O k-NN do Qdrant (HNSW) aproxima vizinhança por navegação em grafo — é um kernel trick estrutural.

Offline (mesh.rs:44-48): cálculo do stress entre pares de nós usando vetores 768D → stored as scalar in links.stress.

Analogia com Integradores Simpléticos ​

A formulação PPR do T4 admite interpretação como sistema hamiltoniano discreto. Conexão estrutural, não metafórica.

Hamiltoniano de stress ​

O stress da aresta H_e = stress × √(M_s × M_t) é o análogo da energia total do sistema. Assim como o hamiltoniano H(p,q) em mecânica clássica, o stress é conservado ao longo de trajetórias no espaço de fase, embora nunca seja calculado durante a query. A query apenas amostra o campo de fase em pontos específicos.

Equações de Hamilton como fluxo PPR ​

A equação de fluxo s_{k+1} = α·P·s_k é o equivalente discreto das equações de Hamilton:

ḋq/dt = ∂H/∂p   (fluxo no espaço de coordenadas)
ṗ/dt = -∂H/∂q   (fluxo no espaço de momento)

Colchete de Poisson discreto ​

O fluxo PPR s_{k+1} = α·P·s_k admite formulação como colchete de Poisson discreto:

{s, H}_discrete = s_{k+1} - s_k = α·P·s_k - s_k

A matriz P codifica a estrutura simplética do grafo — é o análogo discreto da matriz simplética canônica J aplicada ao campo de stress hamiltoniano. Ao contrário da integração de Euler (que não é simplética e deriva energia), o fluxo PPR preserva a estrutura geométrica do espaço de fase semântico porque nunca calcula H explicitamente — apenas amostra seu gradiente via condutâncias das arestas.

No T4, o score s age como coordenada generalizada e a matriz de adjacência P como tensor de transformação canônica. A condutância w codifica o acoplamento estrutural entre variáveis conjugadas. O fator α desempenha o papel de fator de inércia temporal — controla quanto do momento persiste entre hops.

Seção de Poincaré em k=3 ​

O horizonte fixo de 3 hops é uma seção de Poincaré: em vez de integrar a trajetória completa do sistema (o que convergiria para a distribuição estacionária π_i ∝ degree_i), o sistema amostra o espaço de fase apenas no slice temporal t=3. Isto é análogo a fotografar uma órbita planetária em um instante fixo — a fotografia preserva a estrutura geométrica sem necessidade de simular toda a elipse.

A convergência ao equilíbrio seria a trajetória completa; o PPR de horizonte curto é a seção. A distribuição estacionária π é o análogo do atrator global; s_3 é o ponto na variedade estável do sistema.

A preservação de massa de score (sem ganho ou perda em transferência, apenas redistribuição) é o análogo discreto do teorema de Liouville: o volume do espaço de fase semântico acessível é conservado sob fluxo PPR.

Variedade de espaço de fase discreta ​

O grafo k-NN esparso do T4 é a variedade de espaço de fase discreta sobre a qual o sistema flui. O espaço de Hilbert 768D nunca é explicitamente percorrido durante a difusão — a dimensionalidade alta é comprimida no tensor de stress offline, e a propagação ocorre sobre escalares no grafo. Isto é comparável a um integrador simplético que nunca calcula H explicitamente, apenas usa suas derivadas parciais para avançar a solução.

A geometria riemaniana do espaço de embedding é preservada indiretamente: arestas de alto stress correspondem a regiões de alta curvatura no espaço de fase, e a condutância baixa nessas regiões direciona o fluxo por vizinhanças coerentes.

Random Walk com 3 hops, condutância dependente de idade, redistribuição proporcional ao grau, poda por limiar.

Trabalho Relacionado ​

Andersen et al. (2006) — Local graph partitioning via PPR. Provam que o corte de menor condutância pode ser encontrado explorando uma fração do grafo. Andersen busca o corte ótimo global via PPR local; NEXUS usa o cone de luz truncado para preservar sinal da seed.

Gleich (2015) — Survey PageRank além da web. Documenta a propriedade de dominação por grau em grafos não-dirigidos e as limitações da versão base do PPR. Horizonte fixo + condutância dependente de idade respondem a essas limitações.

NeurIPS 2025 (graph path methods vs. classical RAGs) — Random walks até convergência diluem sinal de seed em grafos scale-free. path-based methods preservam locality informacional em comparação a equilibrium-based retrieval. A mesma conclusão é obtida por NEXUS via análise de distribuição estacionária.

Hamiltonian Graph Networks — Redes que codificam o Hamiltoniano de um grafo diretamente nas camadas. A formulação do stress em mesh.rs:38 como energia Hamiltoniana se aproxima desta linha: a condutância w_q(e) = stress × decaimento é funcional do Hamiltoniano do par de nós; a difusão PPR é a dinâmica resultante no limite de horizonte curto.
