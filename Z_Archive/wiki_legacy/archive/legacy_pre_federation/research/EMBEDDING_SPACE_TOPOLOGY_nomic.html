Topologia do Espaço de Embeddings: nomic-embed-text 768d ​

Data: 2026-05-26
Status: Concluído — 6 experimentos executados e sintetizados
Relevância: Calibração do Gatekeeper Fractal (ADR-0014)

Resumo Executivo ​

O espaço de embeddings do modelo nomic-embed-text (768 dimensões) é uma variedade Riemanniana de curvatura constante, estruturalmente equivalente a uma calota de hipersfera S^767 com raio R≈23. Os pontos não são pontos matemáticos — são cones de resolução com abertura angular proporcional à abstração do conceito. A métrica cosine do Gatekeeper opera corretamente neste espaço, mas o threshold τ deveria ser adaptativo ao tipo semântico do vizinho.

2. Dedução Topológica Passo a Passo ​

Passo 1 — Qual é a variedade? ​

E7: ‖step‖ / dist_cosine ≈ 40× constante
→ embeddings vivem numa hipersfera de raio R≈23 no R^768
(‖v‖ ≈ 23 para todos os pontos — confirmado no probe de ortonormalidade)

E10: cosine entre conceitos ∈ [0.28, 0.73], nunca negativo
→ todos os pontos habitam um hemisfério de S^767 (calota positiva)
→ abertura angular ≈ arccos(0.28) ≈ 73.7° do ponto central

Conclusão parcial:

M ⊆ S^767, restrita a uma calota esférica

Passo 2 — Plana ou curva? ​

E3: holonomia = 0° → curvatura seccional local = 0 (Gauss-Bonnet local)
E4: déficit triangular μ≈0.62 constante → curvatura global ≠ 0

Paradoxo aparente: localmente plano, globalmente curvo?
→ Sim — assinatura de espaço de curvatura constante
→ Em S^n com raio R: déficit triangular = Área_geodésica / R²
→ Para R≈23: déficit esperado ≈ 0.60–0.65 ✓ (consistente com E4)

Conclusão:

κ = 1/R² ≈ 1/529 ≈ 0.0019 (curvatura de Riemann)
Holonomia ≈ 0 porque é efeito de segunda ordem em κ

Passo 3 — Simetria e grupo de isometrias ​

E1: cos(A−B, A+B) = 0 → existe plano médio perpendicular a cada par
→ reflexão pelo plano médio é isometria
→ grupo de isometrias contém Z/2Z para cada par

E2: simetria fwd/bwd perfeita → d(A,B) = d(B,A) confirmado mesmo via intermediários

E6: passos NÃO comutativos (dist≈1.47) → grupo NÃO é abeliano
→ T_AB ∘ T_BC ≠ T_BC ∘ T_AB (esperado em S^n para n≥2)

Conclusão:

Grupo de isometrias ≈ SO(768) restrito à calota
Não-abeliano, contém reflexões, preserva distâncias geodésicas

4. Implicações para o Gatekeeper (ADR-0014) ​

A métrica atual do Gatekeeper:

drift(vizinho, atual) = 1 - cos(L2_vizinho, L0_atual)

Opera na calota S^767 e sofre de:

  • (a) Curvatura global κ≈0.0019 → déficit triangular inevitável
  • (b) Cones de resolução θ_vizinho → vizinhos abstratos têm drift efetivo maior

τ Adaptativo Proposto ​

τ_efetivo(vizinho) = τ_base × (θ_ref / θ_tipo_vizinho)

onde θ_ref ≈ 35° (conceito técnico mais específico observado).

6. Achados Algébricos Chave ​

Identidade de Reflexão (E1) ​

∀ A, B ∈ M:  cos(A−B, A+B) ≈ 0

Interpretação: o vetor diferença é perpendicular ao vetor soma.
Consequência: existe sempre um hiperplano médio que separa A e B simetricamente.

Não-Comutatividade (E6) ​

T_AB ∘ T_BC ≠ T_BC ∘ T_AB
dist(composição) ≈ 1.47 &gt;&gt; 0

Interpretação: "ir de A para B e depois para C" ≠ "ir de B para C e depois para A".
Esperado em SO(n) para n≥3.

Invariante de Norma (E7) ​

‖embed(texto)‖ ≈ 23  (σ ≈ 0.3)
‖step linear A→B‖ / dist_cosine(A,B) ≈ 40×

Interpretação: embeddings vivem numa casca esférica quasi-uniforme.
A norma euclidiana do passo é ~40x a distância cosine — reflexo da hipersfera.
